Reflection and Learnings

Within my journey in this course (CPE 232: Managing Enterprise Servers) I was able to create a workflow using infrastructure as code practices to build an enterprise environment in Oracle VM VirtualBox using Debian and CentOS distribution. That being said, I was able to orchestrate remote servers using SSH which allows me to install and communicate in a secure and remote manner also understand the DevOps culture by utilizing GitHub and leveraging infrastructure as code with Ansible. Furthermore, as the course goes by, I was able to evaluate real-world enterprise scenarios virtually, what are the solutions to an enterprise problem, and what is the best service to use in the industry. In addition, I successfully implemented OpenStack on my virtual machine wherein OpenStack is a cloud service that offers plenty of services. Lastly, I would like to thank my instructor Dr. Jonathan V. Taylar for his nonstop support and kind consideration along the course, shared his expertise in managing enterprise servers, and for making the class environment happy and less pressure
